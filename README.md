## 📌 기능 명세

### 1. 인증 / 계정 관리

| 기능           | 설명                              | 우선순위 |
| -------------- | --------------------------------- | -------- |
| 로그인         | ID/PW 입력 후 JWT 발급            | 🔴 필수  |

| 계정 승인/거절 | 관리자가 신규 계정 활성화         | 🔴 필수  |

### 2. 예약 관리

#### 2-1. 예약 등록

| 기능                | 설명                                        | 우선순위 |
| ------------------- | ------------------------------------------- | -------- |

| 예약 색상 지정      | 캘린더에서 구분용 색상 선택                 | 🟡 중요  |
| 메모 입력           | 예약 관련 특이사항 기록                     | 🟡 중요  |

| 예약 수정        | 등록된 예약 정보 수정    | 🔴 필수  |
| 예약 취소/삭제   | 예약 상태 변경 또는 삭제 | 🔴 필수  |

## 🗓️ 개발 단계

```
1단계  백엔드 — DB 설계 + API 구조 개선
2단계  프론트 — Next.js 15 세팅 + 레이아웃 + 공통 컴포넌트
3단계  기능 개발 — 예약 → 캘린더 → 설문 → PDF 순서
4단계  대시보드 / 통계
5단계  모바일 반응형
6단계  Docker 배포 설정
```
